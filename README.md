#  SLS Block 1 Rocket CAD Model

## Overview

This project is a detailed CAD model of the **SLS (Space Launch System) Block 1** rocket. The model is designed to closely replicate the appearance and major external features of the real SLS Block 1 while remaining suitable for display, educational use, and 3D printing.

The model can be printed as a **single-color** print or as a **multi-color** print, depending on your printer and preferences. It is designed with clean geometry and accurate proportions to produce an impressive finished model.

## Features

* Detailed exterior based on the SLS Block 1
* Accurate overall proportions and major structural details
* Compatible with both single-color and multi-color 3D printing
* Suitable for display, educational projects, and hobbyist collections

## 3D Printing

This model can be printed:

* In a single color for a simple, clean appearance.
* In multiple colors for a more realistic representation of the SLS Block 1.

Choose the printing method that best matches your printer's capabilities and desired finish.
